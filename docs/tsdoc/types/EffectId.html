Type alias EffectId<InputType, ResultType>

EffectId<InputType, ResultType>: symbol & { _inputType: InputType; _resultType: ResultType }

The rx-signals Store uses this type to uniquely identify all of its result effects. An EffectId<InputType, ResultType> does not make any use of the generic parameters itself, but is given these parameters only as a trick to let Typescript infer and thus enforce the correct types. Use the getEffectId function to generate a corresponding ID.
